HTML & CSS

CSS

CSS (Cascading Style Sheets) egy stíluslap nyelv, amelyet a HTML dokumentumok megjelenésének leírására használnak. Segítségével meghatározhatjuk a weboldal elemeinek kinézetét, például a színeket, betűtípusokat, elrendezést és egyéb vizuális aspektusokat.

CSS Bemutatás

A CSS szabályok alapvető szerkezete a következő:

                
    selector {
        property: value;
    }

A fenti kód egy egyszerű CSS szabályt mutat be, amely egy elem (selector) stílusát határozza meg egy tulajdonság (property) és annak értéke (value) segítségével.

CSS Kezdő Szint

A kezdő szintű CSS programozás alapvető stílusok és elrendezések használatát foglalja magában. Például egy egyszerű szöveg színezése és középre igazítása:

                
    p {
        color: blue;
        text-align: center;
    }

A fenti kód egy egyszerű CSS szabályt mutat be, amely a bekezdés (p) szövegét kékre színezi és középre igazítja.

CSS Haladó Szint

A haladó szintű CSS programozás magában foglalja a komplex elrendezések és animációk használatát. Például egy egyszerű rács elrendezés létrehozása:

                
    .container {
        display: grid;
        grid-template-columns: 1fr 1fr 1fr;
        gap: 10px;
    }
    
    .item {
        background-color: lightgray;
        padding: 20px;
        text-align: center;
    }

A fenti kód egy egyszerű rács elrendezést mutat be, amely három oszlopból áll, és az elemek közötti távolság 10 pixel.

CSS Profi Szint

A profi szintű CSS programozás magában foglalja a fejlett technikákat, mint például a CSS változók, flexbox és grid layout használata, valamint a reszponzív dizájn. Például egy egyszerű animáció létrehozása:

                
    @keyframes example {
        from {background-color: red;}
        to {background-color: yellow;}
    }
    
    div {
        width: 100px;
        height: 100px;
        background-color: red;
        animation-name: example;
        animation-duration: 4s;
    }

A fenti kód egy egyszerű animációt mutat be, amely egy div elem háttérszínét pirosról sárgára változtatja 4 másodperc alatt.
