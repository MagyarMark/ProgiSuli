HTML & CSS

HTML Alapok

A HTML (HyperText Markup Language) a weboldalak alapvető építőköve. Segítségével strukturálhatjuk és formázhatjuk a weboldal tartalmát. A HTML elemeket címkék (tag-ek) segítségével határozzuk meg, amelyek általában nyitó és záró címkéből állnak.


Ajánlatos használni egy kódszerkeztőt, például a Visual Studio Code-ot, amely a HTML kódokat egyszerűen szerkeszthet. itt tudod letölteni -> link.

HTML Bemutatás

A HTML dokumentumok alapvető szerkezete a következő:

                
<!DOCTYPE html>
<html>
<head>
    <title>Dokumentum címe</title>
</head>
<body>
    <h1>Ez egy főcím</h1>
    <p>Ez egy bekezdés.</p>
</body>
</html>

A fenti kód egy egyszerű HTML dokumentumot mutat be, amely tartalmaz egy főcímet és egy bekezdést.

HTML Kezdő Szint

A kezdő szintű HTML programozás alapvető elemek és attribútumok használatát foglalja magában. Például egy egyszerű lista létrehozása:

                
<ul>
    <li>Első elem</li>
    <li>Második elem</li>
    <li>Harmadik elem</li>
</ul>

A fenti kód egy egyszerű rendezetlen listát mutat be, amely három elemet tartalmaz.

HTML Haladó Szint

A haladó szintű HTML programozás magában foglalja a formok, táblázatok és beágyazott elemek használatát. Például egy egyszerű űrlap létrehozása:

                
<form action="/submit" method="post">
    <label for="name">Név:</label>
    <input type="text" id="name" name="name">
    <br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    <br>
    <input type="submit" value="Küldés">
</form>

A fenti kód egy egyszerű űrlapot mutat be, amely tartalmaz egy név és egy email mezőt, valamint egy küldés gombot.

HTML Profi Szint

A profi szintű HTML programozás magában foglalja a fejlett technikákat, mint például a HTML5 API-k használata, a webkomponensek és a reszponzív dizájn. Például egy egyszerű videó beágyazása:

                
<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    A böngészője nem támogatja a videó elemet.
 </video>

A fenti kód egy egyszerű videó elemet mutat be, amely egy MP4 formátumú videót játszik le.
