C#

C# alapok

A C# (ejtsd: "C-sharp") egy modern, objektumorientált programozási nyelv, amelyet a Microsoft fejlesztett ki. A C# nyelvet elsősorban a .NET keretrendszerrel együtt használják, és széles körben alkalmazzák asztali alkalmazások, webes alkalmazások, játékok és mobilalkalmazások fejlesztésére.


Ajánlatos használni egy kódszerkeztőt, például a Visual Studio-t, amely a C# kódokat egyszerűen szerkeszthet. itt tudod letölteni -> link.

C# Bemutatás

A C# nyelv szintaxisa hasonlít más C-alapú nyelvekhez, mint például a C++ és a Java. Az alábbi példa egy egyszerű "Hello, World!" programot mutat be C# nyelven:

                
    using System;
    
    class Program
    {
        static void Main()
        {
            Console.WriteLine("Hello, World!");
        }
    }

A fenti kód egy egyszerű C# programot mutat be, amely kiírja a "Hello, World!" üzenetet a konzolra.

Változók és adattípusok

A C# nyelvben különböző adattípusokat használhatunk a változók deklarálásához. Az alábbi példa bemutatja néhány alapvető adattípus használatát:

                
using System;

class Program
{
    static void Main()
    {
      int szam = 42; // Egész szám
      double tortSzam = 3.14; // Tört szám
      string szoveg = "Hello, C#!"; // Szöveg
      bool igaz = true; // Logikai érték

      Console.WriteLine("Egész szám: " + szam);
      Console.WriteLine("Tört szám: " + tortSzam);
      Console.WriteLine("Szöveg: " + szoveg);
      Console.WriteLine("Logikai érték: " + igaz);
    }
}

A fenti kód különböző adattípusokat használ, és kiírja azok értékét a konzolra.

Feltételes utasítások

A C# nyelvben az if, else if és else utasításokat használhatjuk feltételek ellenőrzésére. Az alábbi példa bemutatja ezek használatát:

                
using System;

class Program
{
    static void Main()
    {
        int szam = 10;

        if (szam > 0)
        {
            Console.WriteLine("A szám pozitív.");
        }
        else if (szam < 0)
        {
            Console.WriteLine("A szám negatív.");
        }
        else
        {
            Console.WriteLine("A szám nulla.");
        }
    }
}

A fenti kód ellenőrzi, hogy a szám pozitív, negatív vagy nulla, és ennek megfelelő üzenetet ír ki.

Ciklusok

A C# nyelvben különböző ciklusokat használhatunk, például for, while és do-while. Az alábbi példa bemutatja a for ciklus használatát:

            
    using System;

    class Program
    {
        static void Main()
        {
            for (int i = 1; i <= 5; i++)
            {
                Console.WriteLine("Szám: " + i);
            }
        }
    }

A fenti kód egy for ciklust használ, amely 1-től 5-ig kiírja a számokat a konzolra.
