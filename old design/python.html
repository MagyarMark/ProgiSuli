Python

Python Alapok

A Python egy magas szintű, általános célú programozási nyelv, amelyet könnyű olvasni és írni. A Python egyszerű szintaxisa lehetővé teszi a gyors fejlesztést és a kód könnyű karbantartását.

Python Kezdő Szint

A kezdő szintű Python programozás alapvető szintaxis és egyszerű programok írását foglalja magában. Például egy egyszerű "Hello, World!" program:

                
print("Hello, World!")

A fenti kód egy egyszerű Python programot mutat be, amely kiírja a "Hello, World!" üzenetet a konzolra.

Python Haladó Szint

A haladó szintű Python programozás magában foglalja az objektumorientált programozást, modulok és csomagok használatát, valamint a kivételkezelést. Például egy egyszerű osztály definiálása:

                
class Szemely:
    def __init__(self, nev, kor):
        self.nev = nev
        self.kor = kor
    
def bemutatkozas(self):
    print(f"Szia, a nevem {self.nev})
    print(f"és {self.kor} éves vagyok.")
    
szemely = Szemely("Anna", 25)
szemely.bemutatkozas()

A fenti kód egy egyszerű osztályt definiál, amely egy személy nevét és korát tárolja, és egy bemutatkozó üzenetet ír ki.

Python Profi Szint

A profi szintű Python programozás magában foglalja a fejlett technikákat, mint például a dekorátorok, generátorok, aszinkron programozás és a különböző Python könyvtárak és keretrendszerek használata. Például egy egyszerű dekorátor:

                
def my_decorator(func):
    def wrapper():
        print("Valami történik a függvény előtt.")
        func()
        print("Valami történik a függvény után.")
    return wrapper
    
@my_decorator
def mondj_valamit():
    print("Hello!")
    
mondj_valamit()

A fenti kód egy egyszerű dekorátort mutat be, amely egy függvény előtt és után hajt végre műveleteket.
